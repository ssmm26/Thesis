Job Description Overview

Calypso Technology is seeking is seeking a Post-Sales Senior Technical Analyst with relevant work experience in the capital markets or at a financial software company. The ideal candidate will have strong Java development skills with enterprise application experience.

Responsibilities

  • In-depth, hands-on involvement directly with customers implementing the Calypso system. A key responsibility will be working directly with the customer project team implementing the Calypso system, and assist them in implementing their users\u2019 requirements. The technical analyst will then typically follow up with the users to test the implementation, achieve signoff, and continue to move the system toward go-live.
  • Assist developers in understanding the API required to extend the Calypso system, add data to the Calypso system, and build interfaces between Calypso and external systems.
  • Train users and customer developers on the basic use of the system, particularly with the technical setup.
  • System installation, including Java JRE download and installation, relational database installation, data model installation and upgrade scripts, setup and configuration of event-driven framework, configuration of real-time market data feed, etc.
  • Work hands on with the system, configuring it and testing that it meets the customer\u2019s technical requirements.
  • Work with Calypso business analysts, Calypso core development, or customer development to code prototypes of enhancements to ensure they will meet business requirements.
  • Some technical project management work will be expected. In particular, managing the technical issues and tracking the timeline of technical deliverables, in order to keep the customer informed of progress. Assist customers with development of their own internal project plan, including identification of key setup and configuration tasks, scope considerations, timeline and scheduling, and resource requirements.
